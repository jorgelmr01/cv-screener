# CV Screener

Bienvenido a CV Screener, tu asistente inteligente para la selección de candidatos.

## Requisitos Previos

Para ejecutar esta aplicación, necesitas tener instalado **Node.js** en tu computadora.
Si no lo tienes, puedes descargarlo gratis aquí: [https://nodejs.org/](https://nodejs.org/)

## Cómo Iniciar

1.  **Doble clic** en el archivo `INICIAR_APP.bat`.
2.  Se abrirá una ventana negra (terminal) que instalará las dependencias necesarias (solo la primera vez) y luego iniciará el servidor.
3.  Automáticamente se abrirá tu navegador web con la aplicación.

## Solución de Problemas

*   **Si la ventana se cierra inmediatamente:** Asegúrate de haber instalado Node.js.
*   **Si el navegador no se abre:** Abre tu navegador y visita `http://localhost:5173` manualmente.

## Notas Técnicas

Esta aplicación utiliza tecnologías web modernas (React + Vite) que requieren un servidor local para funcionar correctamente. Por eso no es posible abrir directamente el archivo `index.html`. El script `INICIAR_APP.bat` se encarga de todo esto por ti.
